Im Kontext des **Ultimate Master Guard V10** (Pinetree Outbreak Script) beziehen sich diese Begriffe auf das **Sector Rotation Module**, welches analysiert, wie gut eine Aktie zur aktuellen Phase des Wirtschaftszyklus passt.

Hier ist die genaue Bedeutung laut den Quellen:

### **Smart Money**
*   **Definition:** Dieser Status wird angezeigt, wenn der Sektor der Aktie in der **nächsten** Phase des Wirtschaftszyklus begünstigt ("favored") sein wird.
*   **Bedeutung:** Es signalisiert eine **frühzeitige Positionierung**. Der Begriff lehnt sich an die Beobachtung an, dass institutionelle Anleger (das "Smart Money") Sektoren oft schon dann kaufen, bevor die entsprechende wirtschaftliche Phase offiziell beginnt.
*   **Darstellung:** Im Dashboard wird dies blau markiert.

### **Rotating In**
*   **Definition:** Dieser Begriff wird verwendet, wenn der Sektor der Aktie für die aktuelle Phase begünstigt ist **und** der Markt gerade erst in diese Phase übergeht ("transitioning into this phase").
*   **Bedeutung:** Dies gilt als **idealer Einstiegszeitpunkt** ("smart entry"), da der Sektor gerade beginnt, vom konjunkturellen Rückenwind des Zyklus zu profitieren.
*   **Darstellung:** Im Dashboard wird dies grün markiert.

### **Hintergrund: Wie das Script diese Zustände erkennt**
Das Script nutzt ein **Phasen-Scoring-System**, das die relative Stärke von sechs verschiedenen Sektor-ETFs (z. B. Technologie, Finanzen, Energie) vergleicht. Durch einen **Dual-Timeframe-Ansatz** (Vergleich von 60-Tage- und 20-Tage-Trends) erkennt das System, ob eine Phase stabil ist oder ob gerade ein Wechsel (Transition) stattfindet. Wenn eine solche Transition erkannt wird, gleicht das Script dies mit dem Sektor der Aktie ab, um Labels wie "Rotating In" oder "Smart Money" zu vergeben.
